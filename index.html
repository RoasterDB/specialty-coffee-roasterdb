§ 04 · Frequently Asked Questions

Structured Knowledge Base

Dataset & Taxonomy FAQ

What is the RoasterDB Specialty Coffee Dataset?

RoasterDB is a structured, factual database containing over 8,000 specialty coffee releases from 280+ artisan roasters across 20+ countries. Every entry includes standardized attributes like origin country, altitude (masl), process method, varietals, and tasting notes mapped directly to the SCA Flavor Wheel.

How are tasting notes mapped to the SCA Flavor Wheel?

RoasterDB converts poetic or unstructured roaster descriptions (such as "Grandma's peach cobbler" or "bright clementine acidity") into strict 3-tier hierarchical SCA Flavor Wheel paths (for example: Fruity > Other Fruit > Peach). This enables precise semantic filtering and machine learning feature extraction.

What database formats are included in the download?

The download package includes a normalized relational SQLite database file (.db) with separate tables for roasters, coffee beans, and flavor node linkages, alongside clean CSV and JSON exports for direct import into pandas, R, Python scripts, or web applications.
